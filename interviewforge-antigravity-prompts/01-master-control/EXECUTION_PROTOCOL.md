# Antigravity Execution Protocol

Use this short prompt before each individual implementation task after the master prompt.

```text
Work in the existing InterviewForge AI repository. First inspect the current implementation, tests, migrations, and uncommitted changes relevant to this task. Do not rewrite unrelated code.

Implement only the requested vertical slice. Preserve existing behavior, use the current stack and conventions, keep frontend/backend contracts synchronized, and add production-safe validation, authorization, loading/error/empty states, structured AI handling, and tests.

Before editing, state: (1) files to inspect, (2) implementation plan, (3) acceptance criteria, and (4) verification commands. After editing, run the focused checks and report: changed files, API/schema changes, test results, known limitations, and the next safe step. Do not claim completion for mocked or unverified behavior.
```
