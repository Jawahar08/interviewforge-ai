# Contributing to InterviewForge AI

First off, thank you for considering contributing to InterviewForge AI! It's people like you that make open source such a great community.

## Getting Started

1. Fork the repository
2. Create your feature branch (`git checkout -b feature/amazing-feature`)
3. Set up the local environment (Follow the README Quickstart)
4. Commit your changes (`git commit -m 'feat: Add some amazing feature'`)
5. Push to the branch (`git push origin feature/amazing-feature`)
6. Open a Pull Request

## Coding Standards
* We use Prettier for all frontend `.js/.jsx/.css` files.
* Please ensure all Java code compiles via `mvn clean install`.
* Ensure no secrets (API keys) are committed.
